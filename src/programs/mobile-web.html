Mobile Web
Specialist

Este curso va dirigido a personas que ya tienen tiempo trabajando en la industria y están interesadas en aprender tópicos avanzados de la web : testing (mocha, chai, sinon, jest), progressive web apps, accesibilidad, performance y arquitectura. En este curso nos enfocamos en la construcción de aplicaciones usando webpack para construir aplicaciones con frameworks, como volver aplicaciones progressive y el uso de las Chrome Dev Tools para análisis y depuración de aplicaciones web.

Objetivo

Dominar aspectos avanzados del desarrollo frontend como: entendimiento del funcionamiento del navegador a bajo nivel(critical rendering path), automatización de workflow de desarrollo (gulp, webpack), arquitectura y construcción de aplicaciones usando componentes (react) y progressive web apps.

Perfil del participante la salir del programa:

Serás capaz de crear tu propio workflow de desarrollo, construir aplicaciones usando componentes en React y resolver problemáticas asociadas al aspectos técnicos del ecosistema web (performance, accesibilidad), además de entender el background de como crear progressive web apps
