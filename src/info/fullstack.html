FullStack JS

Objetivo

Entender las tecnologías web para poder crear proyectos dinámicos desde 0.

  • FRONTEND: La parte de la página que interactúa con el usuario y vive en el navegador.
  • BACKEND: La parte que maneja los datos, permisos e información y suele vivir en el servidor.

Con esto podrás diseñar una estructura de datos compleja y desarrollar páginas web dinámicas e interactivas que ofrecen una buena experiencia a los usuarios y que sean 100% personalizables.

Perfil del aspirante

Este curso va dirigido a personas con poco o nulo conocimiento del desarrollo web pero que estén interesadas en poder crear páginas o plataformas; Es una gran opción si buscas hacer un cambio de carrera o mejorar tus habilidades para crear sitios complejos para proyectos personales o de negocio.

Al salir podrás:

Analizar problemáticas complejas, crear un modelo de datos, APIs y aplicaciones frontend bajo el conjunto de tecnologías usadas en el MERN stack (MongoDB, Express, React, Node). con todo esto podrás llevar proyectos pequeños por tu cuenta o acoplarte a equipos de desarrollo complejos.

Plan de Estudios

+ Github

Esencial para cualquier desarrollador, es una herramienta que nos ayudará a generar versiones para respaldar y ordenar nuestro código además de ayudarnos a trabajar en equipo de una manera cómoda.

  • Repositorios para proyectos
  • Commits y manejo de versiones
  • Branchs y trabajo en equipo
  • Gitflow

+ Javascript

Es un lenguaje de programación que ha ganado mucha popularidad y debido a su capacidad de poder adaptarse para diversos usos o entornos. Entenderlo a fondo nos abrirá la posibilidad de utilizar diferentes frameworks y dejar de ver la programación como mágia.

  • Paradigmas de programación en JS
  • Javascript (ES6 y posterior)
  • Asíncronia en JS

+ Frontend

Se refiere a la parte de la página que interactúa con el usuario, abarca desde características visuales como los colores hasta interaciones complejas y efectos especiales que dejaran atónitos a los internautas. El frontend también se preocupa por que la página cargue rápido y tenga compativilidad con los diferentes navegadores.

  • HTML Y CSS
  • CSS frameworks y prepocesadores
  • DOM (Document Object Model)
  • Javascript (ES6 y posterior)
  • Peticiones y asincronía
  • Optimización
  • React
  • Redux

+ Backend

La parte de la página que se encarga de manejar la información, permisos y relaciones. Aquí veremos como guardar información en una base de datos, crear usuarios y definir la lógica que nuestra plataforma necesite para funcionar. El blackend vive en los servidores, por lo que aprenderemos como configurar uno y montar nuestra página.

  • Node JS
  • Express
  • MongoDB
  • Modelado de datos
  • Rest APIs
  • Servidores
