Cloud Architect

Objetivo

Construir, desplegar, mantener y monitorear aplicaciones críticas y de alta disponibilidad en GCP o AWS

Perfil del aspirante

Curso dedicado a desarrolladores experimentados para trasladar sus conocimientos a Arquitecturas Cloud y microservicios pasando por las principales tecnologías e infraestructuras vigentes como Docker, Kubernetes, GCP o AWS

Al salir podrás:

Entender las distintas herraminetas disponibles en la nuve y como aprovecharlas al máximo para manejar cualquier proyecto y hacerlo escalable.

Duración

El programa Cloud Architect tiene duración de 2 meses (120 horas).
Las clases son presenciales de Lunes a Viernes de 7:00PM a 10:00PM.
